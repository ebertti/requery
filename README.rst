requery
=======

Simple way to store and use querys in database for use of DBA's

Installation
------------

1. ``pip install requery``

2. Add ``requery`` to your ``INSTALLED_APPS`` in ``settings.py`` ::

        INSTALLED_APPS = (
            # ...
            'requery',
        )

3. Run ``python manage.py migrate``
